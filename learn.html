Rpwn

Program Description

Rpwn is a tool designed for Wi-Fi analysis and attacks. Its main function
is to identify all devices connected to a target Wi-Fi network
and disconnect them using a deauthentication attack.
This tool is part of the Rasmnout Tools suite and is intended
for security testing purposes.

Features

1. Detect all devices connected to the target Wi-Fi network.
2. Perform deauthentication attacks to disconnect all connected devices.
3. Real-time display of detected devices.

Usage

python rpwn.py [-h] -i INTERFACE -Tb TARGET_BSSID [-c CHANNEL] [-v]

Arguments and Options

1. -h, --help
Displays help information and exits the program.
Example:
python rpwn.py -h
2. -i INTERFACE, --interface INTERFACE (required)
Specifies the network interface to use for scanning and the attack.
The interface must be in monitor mode.
Example:
python rpwn.py -i wlan1mon
3. -Tb TARGET_BSSID, --target-bssid TARGET_BSSID (required)
Specifies the target Wi-Fi network using its BSSID.
Example:
python rpwn.py -Tb 00:11:22:33:44:55
4. -c CHANNEL, --channel CHANNEL (optional)
Specifies the channel of the target Wi-Fi network.
Defaults to the current channel if not provided.
Example:
python rpwn.py -c 6
5. -v, --verbose (optional)
Enables verbose mode for detailed output.
Example:
python rpwn.py -v

Example Commands

Find devices on a target Wi-Fi network and disconnect them:
python rpwn.py -i wlan1mon -Tb 00:11:22:33:44:55 -c 6
Verbose mode for detailed tracking:
python rpwn.py -i wlan1mon -Tb 00:11:22:33:44:55 -c 6 -v

Warnings

Using this tool on networks without explicit permission is illegal.
This program is intended only for testing your own networks
or networks you have permission to test. Use responsibly!
