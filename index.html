Introduction

Rasmnout is an integrated toolset for managing and optimizing Bluetooth, Wi-Fi, and network operations. Whether you’re a beginner or an experienced professional, Rasmnout simplifies wireless device management, network analysis, and security tasks. With its intuitive interface and powerful features, Rasmnout supports a wide range of hardware, making it the ideal choice for IoT projects, network testing, and wireless device management.

Features

  • Bluetooth Device Management: Seamlessly connect to and manage Bluetooth devices, perform scans, and interact with nearby devices for a variety of tasks.
  • Wi-Fi Network Scanning & Analysis: Detect and analyze available Wi-Fi networks, perform penetration testing, and monitor network health with ease.
  • Network Monitoring & Attacks: Execute network-based attacks, analyze traffic, and simulate security threats for penetration testing and vulnerability assessments.
  • Advanced Tools for Professionals: Utilize powerful features like FoxWB+ for complex attack simulations and network security evaluations.
  • User-Friendly Interface: Designed for ease of use, allowing both beginners and experts to quickly perform essential tasks without steep learning curves.
  • Cross-Platform Compatibility: Fully compatible with popular hardware like Raspberry Pi (1-5, Zero models) and ESP32/ESP8266 for flexibility in embedded and IoT projects.
  • Customizable for Your Needs: Tailor Rasmnout to your specific use case with its modular tools and advanced configuration options.

Installation of Ravix Connecter and Ravix Connecter Console

Ravix Connecter for Windows and Linux

To install Ravix Connecter for Windows and Linux, you can use one of the following sources:

  • Official Ravix GitHub page
  • Official Rasmnout website

Installation on Linux

To install on Linux, follow these steps:

  1. Update the package list and install the necessary tools:
    2. sudo apt update
    sudo apt install python3 git python3-pip
  2. Download the ravix-con.py file from GitHub:
    3. wget https://raw.githubusercontent.com/rasmnout/ravix/main/con.py -O ravix-con.py
  3. Run the console application with the -h switch to display help:
    4. python3 ravix-con.py -h

Installation on Windows

To install on Windows, follow these steps:

  1. First, download and install Python from the official page: https://www.python.org/
  2. Open PowerShell and download the ravix-con.py file:
    3. wget https://raw.githubusercontent.com/rasmnout/ravix/main/con.py -O ravix-con.py
  3. Run the file using Python and use the -h switch to display help:
    4. python ravix-con.py -h

Ravix and Ravix+ for Raspberry Pi 3/Zero

Installation of Ravix OS can be found on the Installation page and more information and tutorials are available on GitHub:

  • Installation Guide
  • Official Ravix GitHub page

Linux

To install Ravix on Raspberry Pi, follow the steps below:

For Raspberry Pi 3:

wget https://raw.githubusercontent.com/rasmnout/ravix/main/ravixRPI3.img -O ravix.img

For Raspberry Pi Zero:

wget https://raw.githubusercontent.com/rasmnout/ravix/main/ravixRPI0.img -O ravix.img

Next, write the image to your SD card:

sudo dd if=ravix.img of=/dev/ bs=4M status=progress conv=fsync

After writing the image to the SD card, add the file main.conf to the SD card.

In the main.conf file, add the following configuration:

wifi:{
	ssid=""
	psk=""
}
}

If you are using an open Wi-Fi network, use:

wifi:{
	ssid=""
	OPEN
}

Windows

To install Ravix on Windows, download and run the Rasmnout Setup from:

  • Rasmnout Installation Guide

Extract and run the Rasmnout Setup installer. After installation, you should see the Rasmnout Setup icon on your desktop.

Open Rasmnout Setup, select your preferred options and operating system.

Then, run Rasmnout Connecter, and it should connect to Ravix. If Ravix does not connect, it will create a Wi-Fi network named: Ravix-RPI with the Wi-Fi password: R7SMN0U9r7sv1x

Once you connect to this Wi-Fi, you should successfully connect to Ravix!
