Introduction

Rasmnout Tools is a suite of tools designed for analyzing and testing Wi-Fi
and Bluetooth network technologies. This toolset includes operating systems
like RtornW for Wi-Fi and RtornB for Bluetooth, specifically designed for the
Raspberry Pi platform. These operating systems allow you to perform various
network tests and scans, making them ideal for hackers, security experts,
and researchers in the field of cybersecurity.
RtornW is an operating system optimized for Wi-Fi network analysis, enabling
you to carry out Man-In-The-Middle attacks, sniffing, and other security tests.
On the other hand, RtornB focuses on Bluetooth and allows you to analyze
Bluetooth communications, perform remote attacks, and assess the vulnerabilities
of Bluetooth devices.
All tools within Rasmnout Tools are written in Python 3, making them easily
modifiable and extendable. For data visualization and user interaction, the toolset
integrates with the Waveshare 1.3inch OLED Hat, which is connected to
the Raspberry Pi, providing effective real-time information display. This platform
is perfect for performing tests and analyses on the spot.
If you are looking for a simple yet powerful toolset for Wi-Fi and Bluetooth
security analysis, Rasmnout Tools is a great solution for your needs.




Requirements

To run Rasmnout Tools, the following system and software requirements are necessary:

Hardware:

Raspberry Pi (model 3, 4, or Zero WH) for running the RtornW and RtornB operating systems.
Waveshare 1.3inch OLED Hat for displaying output and interacting with the tools in real-time.
Wi-Fi and Bluetooth connectivity for communication and testing.

Software:

Operating System: The tools are fully compatible with Linux (preferably Raspberry Pi OS distribution)
and Windows. Python 3.8+: Required for running all tools and operating systems.
Python is the core for communication and scripting the tools.

Libraries:

PySerial for serial communication (if using external devices).
Bleak for Bluetooth communication and scanning.
Scapy, Wireshark, or other network analysis tools (depends on the specific tool).
Raspberry Pi specific libraries for controlling the display (e.g., Waveshare).

Additional Requirements:

Administrator Privileges: For certain functions, such as scanning network traffic
and interacting with hardware, administrator rights will be needed.
Dependencies: Before installation, you may need to install the required libraries and packages
through pip or other package managers.
If your system meets these requirements, you can easily install and start using Rasmnout Tools
for testing Wi-Fi and Bluetooth networks and devices.
